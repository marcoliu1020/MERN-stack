# MERN-stack

## Installation
1. create /backend/.env file
   - enter MongoDB uri and jsonwebtoken secret
   ```
   PORT=4000
   MONGO_URI=mongodb...
   SECRET=...
   ```

2. cd backend
   ```
   npm install
   ```

3. cd frontend
   ```
   npm install
   ```

## Usage
1. cd backend
    ```
    npm run dev
    ```

2. cd frontend
    ```
    npm run start
    ```
